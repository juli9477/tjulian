User Interface plays a huge role in web design, the success of a website can depend greatly on how the UI is designed. One popular UI design program is Sketch. Sketch is very convenient and includes wireframes, diagramming, pixel-perfect mock ups and illustration. Another well known program is Illustrator. This program makes illustrations and vector art with precise and advanced typography and drawing tools. Illustrator is very versatile, you can create anywhere from marketing material, brand identity, graphics, film and web. On other program that is a classic is Photoshop. Designers typically use photoshop to build web graphics, UI elements, functional mockups and wireframes. Other popular programs include Indesign, Qt Quick, UXPin, Lucidchart, Marvel etc.
