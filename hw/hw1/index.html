Definition

Interactive Design is field of study that focuses on user experience. It centers around the communication of media through cooperative processes between people and technology. Interactive design aims in creating simple, defined target and clear direction and understandable screen interface.

Six Elements

Interactive Design contains six main elements, user control, responsiveness, real-time interactions, connectedness, personalization and playfulness.

Why it is important

Interactive design is needed to show software experiences to users in a way that is understable, useful, consistent, desirable and a meaningful experience. It ensure we are filling the users needs.
